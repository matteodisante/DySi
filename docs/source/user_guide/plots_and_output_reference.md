### 4.1 Thrust Curve

**File:** `thrust_curve.png`

**RocketPy Documentation:**
> "Plots thrust of the motor as a function of time."

**Plot Details:**
- **X-axis:** Time (seconds) from motor ignition to burnout
- **Y-axis:** Thrust force (Newtons)
- **Data Source:** Motor thrust curve file (`.eng` format)

ðŸ¤– **What to Look For:**
- **Smooth curve:** No unrealistic spikes or discontinuities
- **Peak thrust:** Maximum thrust value (labeled on plot)
- **Burn duration:** Time from ignition to burnout (labeled)
- **Total impulse:** Area under curve (labeled, in NÂ·s)
- **Thrust profile:** Progressive (slow start), neutral (constant), or regressive (fast start)

ðŸ¤– **Physical Interpretation:**
- **High peak thrust:** High initial acceleration, risk of structural loads
- **Long burn time:** Sustained acceleration, higher apogee potential
- **Total impulse:** Direct measure of motor power (e.g., "M" class = 5120-10240 NÂ·s)

ðŸ¤– **Common Issues:**
- Spikes in curve: Bad thrust data, check source file
- Very short burn: Check if correct motor file loaded
- Zero thrust: Motor file not found or empty

**Example Annotations:**

### 5.3 Static Margin vs Time

**File:** `static_margin_vs_time.png`

**RocketPy Documentation:**
> "Plots static margin of the rocket as a function of time."

**Plot Details:**
- **X-axis:** Time (seconds)
- **Y-axis:** Static margin (calibers = body diameters)
- **Formula:** Static margin = (CP - COM) / (2 Ã— radius)

ðŸ¤– **What to Look For:**
- **Always positive:** Rocket is stable (CP behind COM)
- **> 1.0 caliber:** Good stability margin (recommended minimum)
- **> 2.0 calibers:** Very stable (may weathercock in wind)
- **Variation:** Changes as COM shifts during motor burn

ðŸ¤– **Physical Interpretation:**
- **Stability Criterion:** SM > 0 for static stability
- **Practical Minimum:** SM > 1.0 for safe flight
- **Optimal Range:** 1.5-2.5 calibers
- **Too High (>3.0):** Excessive stability, weathercocking, reduced altitude

ðŸ¤– **Common Patterns:**
- **Increasing:** COM shifts aft as motor burns (front-heavy motor)
- **Decreasing:** COM shifts forward (aft-heavy motor)
- **Nearly constant:** Well-balanced motor design

ðŸ¤– **Warning Signs:**
- SM < 1.0: Marginally stable, risky
- SM < 0: Unstable, DO NOT LAUNCH
- Large variations: Check motor placement and grain design

## 8. How to Interpret Results

### 8.1 Validation Workflow

**Step 1: Console Summary**
- Check apogee, max velocity, stability margin
- Verify no error messages

**Step 2: State Files**
- Verify initial state parameters correct
- Check static margin > 1.0 calibers
- Confirm fin/nosecone positions are numeric (not N/A)

**Step 3: Motor Plots**
- Thrust curve smooth and reasonable
- Total impulse matches motor spec
- Mass evolution shows complete burn

**Step 4: Rocket Plots**
- Static margin always positive
- Drag curve reasonable (check transonic peak)
- Rocket schematic shows COM ahead of CP

**Step 5: Trajectory Plots**
- Altitude profile smooth
- No unexpected velocity spikes
- Impact velocity < 10 m/s (with parachutes)

### 8.2 Common Issues and Solutions

| Issue | What to Check | Solution |
|-------|---------------|----------|
| Apogee too low | Drag curves, motor thrust | Check drag data, verify motor file |
| Apogee too high | Mass too low, drag too low | Verify rocket mass, check drag curves |
| Unstable (SM < 1.0) | Fin size/position, COM/CP | Move fins aft, increase fin span, move motor forward |
| High impact velocity | Parachute cd_s too small | Increase parachute size or cd_s value |
| Excessive drift | Wind speed, parachute size | Reduce main chute size, deploy lower |
| Unrealistic max velocity | Drag curves, motor thrust | Verify input data accuracy |

### 8.3 Input-Output Relationships

| Input Parameter | Primary Effect On | Secondary Effects |
|-----------------|-------------------|-------------------|
| `rocket.mass` | Apogee altitude (-), max acceleration (-) | Flight time, landing location |
| `rocket.radius` | Drag force (+), static margin units | Max velocity (-), apogee (-) |
| `fin.span` | Center of pressure position (-), stability (+) | Drag (+), max velocity (-) |
| `motor.thrust_source` | Max velocity (+), apogee (+), acceleration (+) | Flight time, drift |
| `parachute.cd_s` | Descent rate (-), impact velocity (-) | Landing drift (+), flight time (+) |
| `environment.wind` | Lateral drift (+) | Landing location, apogee deployment location |

**Legend:** (+) = increases with parameter, (-) = decreases with parameter
