Overview
--------

The technical documentation covers:

- **Output Interpretation**: How to read and analyze simulation plots
- **Rocket Physics**: Aerodynamics, stability theory, flight dynamics
- **Simulation Methods**: RocketPy's calculation approaches
- **Advanced Concepts**: Detailed explanations of parameters and metrics

When to Read This Section
--------------------------

Use the technical documentation when you need to:

✅ **Understand the math** behind stability calculations
✅ **Choose between different metrics** (static vs stability margin)
✅ **Optimize rocket design** with deep understanding
✅ **Debug unexpected results** by understanding the physics
✅ **Extend the framework** with custom implementations

For step-by-step instructions, see :doc:`/user/tutorials/index` instead.
