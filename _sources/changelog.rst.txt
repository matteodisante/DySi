# Changelog

All notable changes to rocket-sim will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-12

### Added
- Complete documentation restructuring with professional Sphinx setup
- Getting Started guide with installation, quickstart, key concepts, and next steps
- Publication-quality documentation structure following NumPy/SciPy standards
- Comprehensive configuration validation system
- Motor state export with 35+ attributes and 11 curve plots
- Air brakes control with PID, bang-bang, and MPC controllers
- Weather integration (Wyoming, GFS, ERA5)
- Complete test suite

### Changed
- Documentation completely rewritten for clarity and usability
- Improved user experience with clear navigation paths
- Enhanced cross-referencing throughout documentation

### Fixed
- Documentation build warnings resolved
- Improved error messages with actionable suggestions

## [Unreleased]

### Planned
- Monte Carlo uncertainty quantification
- Multi-stage rocket support
- Advanced reporting system
- Additional tutorial content
- How-to guides for common tasks
