What is rocket-sim?
-------------------

**rocket-sim** is a production-ready simulation framework built on `RocketPy <https://github.com/RocketPy-Team/RocketPy>`_,
designed for:

* **Student rocketry teams** designing competition vehicles
* **Aerospace engineering students** learning trajectory simulation
* **Research groups** analyzing rocket performance
* **High-power rocketry enthusiasts** planning ambitious flights
